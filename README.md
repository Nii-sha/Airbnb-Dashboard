# Airbnb-Dashboard
**1. Title**
**Website Traffic & User Engagement Analysis Dashboard**

**2. Short Description (2–3 lines)**
This Power BI dashboard analyzes website traffic and user engagement patterns across different traffic sources and time periods. It helps identify high-performing channels, user behavior trends, and key engagement metrics to support data-driven marketing decisions.

**3. Tech Stack**

* **Power BI** – Data visualization and dashboard creation
* **Power Query** – Data cleaning and transformation
* **DAX** – Calculated measures and KPIs
* **Excel / CSV** – Data storage and preprocessing

**4. Data Source**
Website analytics dataset containing information on traffic sources, sessions, users, page views, bounce rate, and engagement metrics collected from web analytics platforms.

**5. Features / Highlights**

**Business Problem**
Organizations need to understand which traffic sources drive the most engagement and how users interact with the website to improve marketing performance.

**Goal of the Dashboard**
To monitor website performance, identify high-traffic channels, and analyze user engagement trends for better marketing and content strategies.

**Walkthrough of Key Visuals**

* KPI cards showing total users, sessions, and page views
* Traffic source distribution to identify top channels
* Time-based trends showing changes in user engagement
* Comparative visuals highlighting performance across sources

**Business Impact**
The dashboard enables businesses to optimize marketing strategies, focus on high-performing traffic channels, and improve overall website engagement and conversion potential.
